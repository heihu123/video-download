# YT-DLP 桌面视频下载器

> 简单 · 安全 · 本地运行 · 支持 YouTube / Bilibili 等 1000+ 网站

还在为找不到好用的视频下载工具而烦恼？  
**YT-DLP 桌面下载器**来了！

这是一款完全在你电脑上运行的轻量级桌面程序，无需联网服务器，不上传任何数据，保护你的隐私与安全。只需粘贴视频链接，点击“下载”，即可将高清视频保存到本地！

## ✅ 核心特点

- **一键下载**：粘贴链接 → 点击按钮 → 自动完成  
- **支持音视频合并**：自动调用 FFmpeg，输出完整 MP4 文件  
- **本地运行**：所有操作在你自己的电脑上完成，无网络传输，无隐私泄露  
- **多平台兼容**：基于 Python 开发，当前支持 Windows（后续可扩展 macOS/Linux）  
- **开源透明**：基于强大的开源工具 [yt-dlp](https://github.com/yt-dlp/yt-dlp)

## 📥 如何使用？

1. 下载本程序（包含 `yt-dlp.exe` 和 `ffmpeg.exe`）  
2. 双击运行 `VideoDownloader.exe`  
3. 粘贴任意支持网站的视频链接（如 `https://www.youtube.com/watch?v=...`）  
4. 点击【📥 下载视频】  
5. 完成后点击【📂 打开下载文件夹】查看视频  

> 💡 视频默认保存在程序同级的 `downloads` 文件夹中。

## ⚠️ 注意事项

- 首次运行时，部分杀毒软件可能误报（因 yt-dlp 常被滥用），请放心添加信任。  
- 请遵守各平台的使用条款，仅下载你有权访问的内容。  
- 不支持需要登录的会员专享内容（除非手动配置 Cookie）。
